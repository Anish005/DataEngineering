connectinng and creating a DB connection in postgres

docker run -it \
  -e POSTGRES_USER="root" \
  -e POSTGRES_PASSWORD="root" \
  -e POSTGRES_DB="ny_taxi" \
  -v ny_taxi_postgres_data:/var/lib/postgresql \
  -p 5434:5432 \
  --network=pg-network \
  --name pgdatabase \
  postgres:18


Establishing connection for data ingestion

uv run pgcli -h localhost -p 5434 -u root -d ny_taxi

below command to run the script if not dockerised:

uv run python ingest_data.py \                      
  --pg-user=root \
  --pg-pass=root \
  --pg-host=localhost \
  --pg-port=5434 \
  --pg-db=ny_taxi \
  --target-table=yellow_taxi_trips_2021_1 \
  --year=2021 \
  --month=1

After dockerised, run the following command from the root directory (where docker-compose.yml is located):

docker run -it --rm \
  --network=pg-network \
  taxi_ingest:v001 \
    --pg-user=root \
    --pg-pass=root \
    --pg-host=pgdatabase \
    --pg-port=5434 \
    --pg-db=ny_taxi \
    --target-table=yellow_taxi_trips \
    --year=2021 \
    --month=1
